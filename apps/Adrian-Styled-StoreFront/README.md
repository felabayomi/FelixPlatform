# Adrian-Styled-StoreFront

This app is the Felix Platform build for **Adrian’s Styled Collection**.

## Naming
- **Internal:** `Adrian-Styled-StoreFront`
- **Public:** `Adrian’s Styled Collection`
- **Admin label:** `Adrian Store`

## Platform rule
This storefront must use the shared Felix ecosystem:
- shared backend
- shared Neon database
- shared admin dashboard
- shared auth
- shared email stack
