# WACI Web App

Public website scaffold for **Wildlife Africa Conservation Initiative (WACI)**.

## Platform integration
- Shared Felix backend
- Shared Neon/Postgres data
- Shared admin dashboard
- Shared support/email pipeline

## Domain wiring
- `wildlifeafrica.org` → WACI frontend
- `api.felixplatforms.com` → shared backend
- `admin.felixplatforms.com` → shared admin dashboard

## Public app path
- `apps/WACI/web`

## Key identifiers
- `app_name: WACI`
- `storefront_key: waci`
